# PRD: Сборник Дуа - Islamic Dua Cards App

## Original Problem Statement
Create 8 Islamic dua cards with:
- Arabic text
- Transliteration in Russian
- Russian translation

Based on authentic hadith sources about protection from shaitan, forgiveness prayers, etc.

## User Personas
- Russian-speaking Muslims seeking authentic duas for daily recitation
- Users wanting to learn Arabic pronunciation through transliteration

## Core Requirements (Static)
1. Carousel/slider format for card navigation
2. Copy text functionality for each section
3. Light theme with classic Islamic style
4. Simple cards only, no additional features

## What's Been Implemented (Jan 2026)
- [x] 8 Dua cards with Arabic, transliteration, translation
- [x] Carousel navigation with prev/next buttons
- [x] Dot indicators for quick navigation
- [x] Copy to clipboard functionality with toast notifications
- [x] RTL support for Arabic text with Amiri font
- [x] Islamic color theme: emerald, gold, cream
- [x] Mobile-responsive design
- [x] Card counter showing current position

## Tech Stack
- Frontend: React + Tailwind CSS
- UI Components: Shadcn/UI Carousel
- Fonts: Amiri (Arabic), Playfair Display, Merriweather
